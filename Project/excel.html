Dasboards

Dashboard, in a nutshell, is a Graphical User Interface meant to provide at-a-glance view of KPIs relevant to a business process

Charts

When it comes to graphical representation of data, charts are the way to go. A chart is basically a graphical representation meant for data visualization, in which the data is represented by certain "symbols"

LookUp & Index

In its simplest form, =VLOOKUP() and =INDEX() are Excel functions used when one needs to find things in a table or a range. When combined with =MATCH() Function, they become a useful data tool

Data Validation & Conditional Formatting

Data Validation are the rules meant to control the type of data or the values that users enter into a cell

Excel Formulas

Excel Functions are predefined formulas that perform calculations by using specific values, called arguments, in a particular order

Pivot - Tables & Charts

A PivotTable is an interactive way to quickly summarize large amounts of data. One can use a Pivot Table to analyze numerical data in detail, and answer unanticipated questions about data

Microsoft Excel

Microsoft Excel is a spreadsheet application developed by Microsoft. It features calculation or computation capabilities, graphing tools, pivot tables, and a macro programming language called Visual Basic for Applications. Advanced Excel refers to the features and functions of Microsoft Excel, which helps the user to perform complex calculations, and perform data analysis.
