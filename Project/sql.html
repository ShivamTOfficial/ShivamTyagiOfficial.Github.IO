Basic SQL

A SQL operator is a reserved word / character that is used to perform certain tasks such as comparisons, arithmetics, etc.

SQL Index

A SQL index is a quick lookup table for finding records that users need frequently. It is small, fast, and optimized for quick lookups.

SQL Joins

A join clause in SQL is meant to combine columns from one or more tables into a new table. It is used to combine data or rows from two or more tables based on a common field between them.

CRUD Operations

CRUD: CREATE. READ. UPDATE. DELETE.
These are four vital operations in SQL.

Stored Procedures

A stored procedure is a prepared SQL code that one can save, so that the code can be reused time and time again.
